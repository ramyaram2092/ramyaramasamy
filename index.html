Game Maker

Overview

Developed a game maker that allows the user to build simple arcade games like space invaders, pacman. The main intent of the project was to get hands on experience with design patterns.

Design

The framework was designed to accommodate the MVC pattern. The view is represented in FXML to separate the view from the actual application logic. The controller was designed to accommodate Command design pattern. Observer Pattern was used for the sprite’s properties and behavior updates. The strategy pattern was used to implement the various possible actions that can be added to a sprite.

Responsibilities:

Designed and developed the controller and View portion of the system

Tech Stack:

JavaFX, Java, SonarQube, Object Oriented Programming, Agile, Design Patterns
